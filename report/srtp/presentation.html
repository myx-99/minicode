2. 系统架构

Plan-and-Execute with ReAct sub-loops — execute 和 tools 之间形成自由 ReAct 子循环（LLM 自主决定 0-N 次工具调用），步骤间按 plan 顺序推进，reflect 评估每步结果。

8. 关键挑战：BUG-001 意图漂移

解决方法 (V3)

教训：正则意图分类有盲区，无法覆盖"你是谁"这种边界 case。V3 从「正则分类」转向「模式+模型驱动」，让 agent 模式自然处理简单对话。同时催生了 Intent Auditor 的研发需求——即使 agent 模式无法 100% 防止误操作，需要专门的审计机制。
